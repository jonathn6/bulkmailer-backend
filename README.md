# bulkmailer-backend

## What this is
this works as a backend go-between, connecting an iOS bulk emailing application with a render.com instance.

## Features
- receives a JSON data structure of array or email addresses, email subject, email body, and attachment and passes it along to render.com

## Tech Used
- javascript

## How to Run
- Called by iOS bulk emailing application

## Notes
- future upgrade will provide real time update of emails sent
